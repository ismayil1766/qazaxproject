Bu versiyada istifadəçi OTP/email təsdiqi çıxarılıb və şifrə bərpası admin sorğusu kimi işləyir.

Nə dəyişdi:
- Qeydiyyat: telefon + şifrə ilə birbaşa hesab açılır.
- Giriş: telefon və ya email + şifrə ilə birbaşa daxil olunur.
- İstifadəçi üçün OTP endpoint-ləri deaktivdir.
- Şifrə bərpası səhifəsi artıq adminə sorğu göndərir.
- Admin paneldə yeni "Şifrə bərpa" bölməsi var.
- Admin yeni şifrə yazıb sorğunu bağlaya bilər.
- Şifrə yenilənəndə istifadəçinin bütün sessiyaları bağlanır.
- Elan yaratmada sadə anti-spam limitləri qalır.

Vacib deploy qeydləri:
- Prisma schema dəyişib. Deploy-dan əvvəl bunları işlət:
  1) npm install
  2) npx prisma generate
  3) npx prisma db push
  4) npm run build
- Yeni model: PasswordResetRequest
- Mövcud istifadəçilər işləyəcək.
- Yeni istifadəçilər üçün email məcburi deyil; sistem daxili email yaradır.


Əlavə qoruma qatları:
- Login üçün həm IP, həm də hesab/nömrə üzrə throttling var.
- Qeydiyyat üçün həm IP, həm də telefon nömrəsi üzrə limit var.
- Şifrə bərpası sorğuları həm IP, həm də nömrə üzrə limitlənir.
- Bütün /api POST sorğuları middleware ilə same-origin yoxlamasından keçir.
- Elan yaratmada IP və user limitləri, duplicate yoxlaması və pending limit qalır.
